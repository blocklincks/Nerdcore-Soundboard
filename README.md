# Nerdcore Soundboard
SoundBoard for song you put inside. There is already some inside

This project is open source. You can use it however you like, i don't care. You don't need to credit me. This project was just for me to learn C++.

#Requirement 
  *Visual studio updated 
    .Windows: Visual Studio 2022 (with "Desktop development with C++" checked) or MinGW-w64 (GCC).
    .macOS: Xcode Command Line Tools (clang).
    .Linux: GCC or Clang (g++ or clang++).
  *download or build SFML 3 binaries that exactly match your compiler version.
  *must link the following SFML modules in their build system:
    .sfml-graphics
    .sfml-audio
    .sfml-window
    .sfml-system
